# Data

This project uses the Auto MPG dataset from the UCI Machine Learning Repository.

Dataset URL:  
https://archive.ics.uci.edu/ml/machine-learning-databases/auto-mpg/auto-mpg.data

The dataset is loaded directly from the UCI repository in the R script, so the raw data file is not stored in this repository.

After preprocessing, rows with missing horsepower values are removed. The final cleaned dataset contains 392 complete observations.
